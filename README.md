## üîß Installation

### Backend Node Server
```bash
git clone https://github.com/adiabb05-1317/Road-Guardian.git
cd Road-Guardian
npm install
npm run dev
```
### Frontend Server
```bash
cd path/to/frontend
npm install
npm start
```
### Python Flask Server for YOLOv5 Model
```bash
cd path/to/flask_server
pip install -r requirements.txt
gunicorn main:app --reload
```
